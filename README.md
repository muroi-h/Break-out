# Break-out

## コンセプト

今までとは少し違ったブロック崩し。

## 使用アプリケーション

- Unity
- Visual Studio
- Metasequia4
- Blender

## 制作期間

約2ヶ月

## 工夫した点

### アイテムブロック

ブロックを壊すとランダムでアイテムが出現するアイテムブロックを導入。
取ると自機の幅が拡大。

### 色ごとにブロックの強度を変化

色ごとにブロックの強度が変化するシステムを導入。
当たるごとにブロックの色が薄くなり、青は1回、赤は2回、緑は3回で壊れるように設定。

### ブロックを長方形から六角形へ

プレイヤーに楽しんでいただけるよう2ステージ目のブロックを長方形から六角形へ変更。
そのすることによって、長方形に比べて六角形は面が多いため跳ね返る角度の予測が困難に。
